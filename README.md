# Pymath
Math Made Simple

Pymath is a simple package to deal with numbers.

**Install**

import os

os.system("pip install Pymath")

import Pymath

**Add**

pymath.add(a, b)

**Subtract**

pymath.sub(a, b)

**Multiply**

pymath.mul(a, b)

**Divide**

pymath.div(a, b)
